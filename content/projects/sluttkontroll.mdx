Mitt desidert største prosjekt. Ideen til prosjektet kom da vi ikke hadde et egnet system for utføring av våre sluttkontroller. 

Tanken var et enkelt system med enkle sjekkpunkter, registrering av avvik og målinger. I etterkant av kontrollen skulle vi enkelt få ut komplett rapport. 

Fra start med PHP - og sympony framwork, har prosjektet hatt mange revisjoner. 

- PHP - Symphony framwork
- PHP backend, VUE frontend 
- Node backend, React frontend 


Appen har hele veien blitt laget i React Native og Expo, allikevel har kodebase her også gjennomgått store forandringer, til å bli 100% typescript og bedre håndtering av data.
